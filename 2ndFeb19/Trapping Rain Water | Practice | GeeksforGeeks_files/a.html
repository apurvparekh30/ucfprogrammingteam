                                      I have used dp. Maintain two dp arrays, one dpleft and another dpright. Let dpleft[i] be the maximum elevation map from 1 to i , and let dpright[i] be maximum elevation map from i to n, Then maximum amount of water you can trap on ith position will be max ( min( left[ i ] , right[ i ] ) - a[ i ] , 0 ) .
                                      The sum for each i from 1 to n gives you answer!

                                                          This is O(n) solution:

                                                          int find_res(int A[], int n){

                                                          int sum = 0;

                                                          int s1 = 0;

                                                          int i = 1;

                                                          while(i < n){

                                                          while(A[i] < A[s1]){

                                                          i++;

                                                          }

                                                          for(int k = s1; k < i; k++){

                                                          sum += (std::min(A[s1], A[i]) - A[k]);

                                                          A[k] += (std::min(A[s1], A[i]) - A[k]);

                                                          }

                                                          while(A[s1] <= A[i] && i < n){

                                                          A[s1] = A[i];

                                                          s1++;

                                                          i++;

                                                          }

                                                          }

                                                          return sum;

                                                          }

                                                                                                          Saurav Kumar Sinha 2 months ago

                                                                                                          The simplest solution is the start finding water gathered from index 1 till last but one element. For each of these indices we have to find the max of left Side and max on right side. The min of Max of Left and max of right gives the upper boundary till which water can be collected. Now number of water trapped at that index is minOftwo found earlier - currentHEight. If this is positive it adds water or else it adds 0 meaning we did not trap any water here.

                                                                                                          public class TrappingRainWater {

                                                                                                          public static void main(String ...args) {
                                                                                                          int []input = {2,0,1,4,1,2,4,3,1};
                                                                                                          int numberOfWaterDropsGathered = 0;
                                                                                                          for (int i =1; i < input.length -1; i++) {
                                                                                                          int leftMax = findMax(input, 0, i - 1);
                                                                                                          int rightMax = findMax(input, i + 1, input.length -1);
                                                                                                          int min = leftMax < rightMax ? leftMax : rightMax;
                                                                                                          if (min - input[i] > 0) {
                                                                                                          numberOfWaterDropsGathered += min - input[i];
                                                                                                          }
                                                                                                          }
                                                                                                          System.out.println("Number of dropes gathered = " + numberOfWaterDropsGathered);
                                                                                                          }

                                                                                                          private static int findMax(int []input, int left, int right) {
                                                                                                          int max = 0;
                                                                                                          for (int i =left; i <= right; i++) {
                                                                                                          if (max < input[i]) {
                                                                                                          max = input[i];
                                                                                                          }
                                                                                                          }
                                                                                                          return max;
                                                                                                          }

                                                                                                          }
